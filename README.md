# projeto_front_end

Projeto em HTML/CSS/JavaScript, parte da avaliação na disciplina de Programação Web Front-End na UTFPR

## Etapa 1

O projeto consiste em um site institucional para uma ONG fictícia, de nome "Leitor Pé Vermelho", inventada unicamente como contexto para a criação do site, assim como dois projetos também fictícios. Além da página principal, foram criadas as páginas de cadastro e login.

Através dessa página, foram aplicados conceitos básicos de HTML e CSS através dos requisitos apresentados pela professora. Estão incluidos entre esses requisitos:
- Uso de tags e estruturas básicas do HTML;
- Múltiplas Páginas;
- Cabeçalho e Rodapé presentes em todas as páginas;
- Menu de Navegação com links funcionais para as seções da página principal;
- Uso de ícones;
- Inserção de imagens;
- Links funcionais externos;
- Paleta de cores;
- Uso de fontes externas;
- Formulário de Login e de Cadastro.

## Etapa 2

Implementação da página "Admin" para aplicar conceitos básicos de JavaScript. Entre as funcionalidades apresentadas, estão:
- Armazenamento local de dados no navegador;
- Formulário para cadastrar usuários;
- Lista dos usuários cadastrados;
- Funcionalidade para pesquisar e excluir usuários.
